Вертикальный ритм

Вертикальный ритм — это концепция, пришедшая из полиграфии. Мы пытаемся сохранить на странице вертикальное пространство между элементами, согласованными друг с другом

Вертикальный ритм

Верстка вертикального ритма — это, в первую очередь, правильная расстановка текста, изображений и других элементов по базовой сетке

Вертикальный ритм

  • размер шрифта (font-size)
  • высота строки (line-height)
  • внутренние и внешние отступы (padding, margin)

Интерлиньяж

Эта разница в виде свободного пространства распределяется сверху и снизу от отображаемых глифов

        p {
            font-size: 18px;
            line-height 1.5; 
        }

Интерлиньяж = (font-size / line-height) / 2

Высота строки

При прочих отсутствующих значениях, высота строчного блока равна значению line-height 

          p {
            font-size: 18px;
            line-height 1.5; 
        }

        p {
          font-size:1em;
          margin-top: 1.5em;
          margin-bottom: 1.5em;
        }

Сетки

Сетки

Сетка — основа современного веб-дизайна. Сетки помогают читателю воспринимать текстовую и визуальную информацию

Сетки

Построим свою сетку

  1. Определяем количество колонок
  2. Регулируем межколоночное расстояние (gutters) и отступы от края экрана (margins)
  3. С помощью вертикального ритма задаем систему отступов

Сетки для адаптивного дизайна

Пример адаптивных сеток:

  1. 12 колонок - десктоп
  2. 8 колонок - планшет
  3. 2 колоноки - мобильные

Модульная сетка

Состоит из прямоугольников с заданной шириной и высотой. Строится на пересечении колонок и строк вертикального ритма

Сетки 8pt и 10pt

  • 8px: подходит для интерфейсов. Пришла из дизайн-системы Google Material Design
  • 10px: подходит для веб-дизайна

Point

1pt = 4/3 физических пикселей

При разрешении «2x» (@2x) 1pt = 4 пикселя, поскольку разрешение удваивается по осям X и Y, что делает его шириной 2 пикселя и высотой 2 пикселя.

Сетка — это стандарт

Если в макете есть сетка, верстальщик быстрее:

  1. разберется с системой пропорций и отступов
  2. настроит поведение контента страницы для разных размеров экрана
  3. соберет адаптивную версию страницы

font-size

font-style

  1. normal — устанавливает для текста обычный шрифт (отключает существующий курсив)
  2. italic — устанавливает для текста курсивную версию шрифта
  3. oblique — устанавливает для текста имитацию курсивного шрифта

font-style

        p {
          font-style: normal;
        }

font-weight

  1. normal, bold — нормальный и полужирный шрифт
  2. lighter, bolder — устанавливает жирность текущего элемента на один шаг легче или тяжелее, чем жирность его родительского элемента
  3. 100–900 — числовые значения жирности

font-weight

        p {
          font-weight: 500;
        }

text-transform

  1. none — отменяет любую трансформацию
  2. uppercase — преобразует весь текст в верхний регистр
  3. lowercase — преобразует весь текст в нижний регистр
  4. capitalize — преобразует все слова так, чтобы первая буква была заглавной

text-transform

        }

text-decoration

  1. none — отменяет все существующие текстовые оформления

text-align

Цвет

На уровне элементов HTML, всему можно присвоить цвет

Цвет для текста

  1. color: применяется к тексту
  2. background-color: цвет фона
  3. text-shadow: тень для текста
  4. text-decoration-color: для элементов оформления текста (подчёркивание, перечёркивание)
  5. text-emphasis-color: используется для выделения диакритических знаков, прилегающих к каждому текстовому символу

Цвет для блоков

  2. background-color: цвет фона блока
  3. column-rule-color: цвет линий, которые разделяют колонки текста
  4. outline-color: цвет контура, вокруг границы элемента

Цвет для блоков

  3. border-block-start-color, border-block-end-color: установливает цвет границ, которые расположены ближе всего к началу и концу блока
  4. border-inline-start-color, border-inline-end-color: установливает цвет границы, расположенной ближе всего к началу и концу текста в блоке

Альтернативный текст (alt)

текстовое описание изображения для использования в ситуациях, когда изображение не может быть просмотрено / отображено или отрисовка занимает много времени из-за медленного интернет-соединения

Альтернативный текст (alt)

Нужен:

  4. для поисковых систем
  5. если у пользователя отключено отображение изображений на странице

Вставка через CSS

Подходит для декоративных изображений. Есть два способа вставить декоративное изображение: сделать фоном или добавить, как псевдоэлемент

background-image

Значения:
  1. none
  2. url('путь_к_изображению')
  3. linear-gradient(), radial-gradient(), repeating-linear-gradient(), repeating-radial-gradient()
        .element {
          background-image: url("image.jpg");
        }

background-repeat

  1. repeat — значение по умолчанию, изображение повторяется по горизонтали и вертикали
  2. repeat-x — повторяется только по горизонтали
  3. repeat-y — повторяется только по вертикали
  4. no-repeat — отображается только один раз без повторения
        .element {
          background-repeat: no-repeat;
        }

background-size

  1. auto — отображается в своём естественном размере
  2. cover — масштабируется так, чтобы полностью заполнить фон элемента
  3. contain — масштабируется так, чтобы полностью поместиться в фон элемента без искажений
        .element {
          background-size: cover;
        }
